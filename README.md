# Dotfiles

This repository contains my personal configuration files for **NixOS**, **Hyprland**, and other tools I use on my system.  
It is organized to make it easy to track, version, and replicate my setup across machines.

---

## Repository Structure

- `nixos/` — NixOS system configuration files (`configuration.nix` and related modules)  
- `hyprland/` — Hyprland Wayland compositor configuration (`hyprland.conf`)  
- `...` — Additional dotfiles will be added as I create them
