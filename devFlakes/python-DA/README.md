# Python Data Science Dev Environment (Nix Flake)

This repository provides a reproducible Python development environment for data analysis and visualization using **pandas**, **Plotly**, **NumPy**, **Matplotlib**, and **Jupyter**, powered by **Nix flakes**. 

---

## 🛠 Features

- Reproducible Python environment via Nix
- Core packages included:
  - `pandas` – data manipulation
  - `numpy` – numerical computation
  - `matplotlib` – classic plotting
  - `plotly` – interactive plots
  - `jupyter` – notebooks
- Compatible with **Niri** + **Alacritty** terminals
